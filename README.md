# Luxe Leathers – Premium Leather Brand Website

## 🌐 Overview
Luxe Leathers is a modern brand website designed for showcasing handmade leather bags for both men and women. The website focuses on luxury branding, elegant UI/UX design, and product presentation to create a premium shopping experience.

## 👜 Features
- Elegant luxury-style homepage
- Product showcase for leather bags
- Men's and women's collection sections
- Responsive design for all devices
- Clean and modern UI/UX
- Brand storytelling section

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Purpose
This project was created to simulate a real-world e-commerce brand website for a luxury leather business and demonstrate my frontend design skills.
